# clash
рЃБрЃюрЃўрЃЋрЃћрЃарЃАрЃўрЃбрЃћрЃбрЃўрЃА рЃърЃўрЃЌрЃЮрЃюрЃўрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃў

рЃћрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА Clash Royale-рЃўрЃА рЃЉрЃљрЃарЃљрЃЌрЃћрЃЉрЃўрЃА рЃЏрЃћрЃюрЃћрЃ»рЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА SQLite рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљрЃќрЃћ рЃЊрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА matplotlib-рЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА.

рЃЏрЃЮрЃЎрЃџрЃћ рЃљрЃдрЃгрЃћрЃарЃљ рЃЦрЃљрЃарЃЌрЃБрЃџрЃљрЃЊ:

рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА Clash Royale-рЃўрЃА рЃЉрЃљрЃарЃљрЃЌрЃћрЃЉрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА рЃЉрЃљрЃќрЃљрЃерЃў рЃЊрЃљ рЃўрЃФрЃџрЃћрЃЋрЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃљрЃА:

­ЪћЇ рЃЉрЃљрЃарЃљрЃЌрЃўрЃА рЃЏрЃЮрЃФрЃћрЃЉрЃюрЃљ рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ

РъЋ рЃљрЃ«рЃљрЃџрЃў рЃЉрЃљрЃарЃљрЃЌрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ

­Ъћё рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃЉрЃљрЃарЃљрЃЌрЃўрЃА рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ

РЮї рЃЉрЃљрЃарЃљрЃЌрЃўрЃА рЃгрЃљрЃерЃџрЃљ ID-рЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ

­ЪЊі рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃарЃљрЃцрЃўрЃЎрЃБрЃџрЃљрЃЊ рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ

рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљрЃерЃў рЃерЃћрЃЊрЃўрЃА:

рЃЉрЃљрЃарЃљрЃЌрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљ рЃўрЃерЃЋрЃўрЃљрЃЌрЃЮрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ (рЃАрЃЋрЃћрЃбрЃўрЃА рЃњрЃарЃљрЃцрЃўрЃЎрЃў)

рЃћрЃџрЃўрЃЦрЃАрЃўрЃарЃўрЃА рЃцрЃљрЃАрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃљ (рЃърЃљрЃў рЃњрЃарЃљрЃцрЃўрЃЎрЃў)

рЃњрЃљрЃЏрЃљрЃарЃ»рЃЋрЃћрЃЉрЃўрЃА рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃљрЃдрЃљрЃџрЃў рЃЏрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃћрЃџрЃў рЃЏрЃЦрЃЮрЃюрЃћ 10 рЃЉрЃљрЃарЃљрЃЌрЃўрЃА рЃЕрЃљрЃЏрЃЮрЃюрЃљрЃЌрЃЋрЃљрЃџрЃў (рЃ░рЃЮрЃарЃўрЃќрЃЮрЃюрЃбрЃљрЃџрЃБрЃарЃў рЃАрЃЋрЃћрЃбрЃўрЃА рЃњрЃарЃљрЃцрЃўрЃЎрЃў)

рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљрЃАрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљрЃА, рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљрЃА рЃЊрЃљ рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљрЃА Python-рЃўрЃЌ.
